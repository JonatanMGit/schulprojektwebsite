## Lokal hosten

Klone das Porjekt

```bash
  git clone https://github.com/JonatanMGit/schulprojektwebsite.git
```

Gehe in den Download Ordner

```bash
  cd schulprojektwebsite
```

Installiere den Webserver

```bash
  npm install
```

Starte den Auto Reload server

```bash
  npm run dev
```

Starte den festen server

```bash
  npm run serve
```
